# Personal AI Telegram Bot 🤖

A multilingual AI Telegram bot powered by OpenAI GPT.  
It can chat in **any language**, auto-reply to messages, and run 24/7.

---

## **Features**
- Chat in **any language** 🌎  
- Auto-reply using OpenAI GPT-4 or GPT-3.5  
- Works in Telegram with `/start` command  
- Ready for cloud deployment (Replit, Render, Heroku, Railway)  
- Easy to customize commands and responses  

---

## **Setup Instructions**

### **1️⃣ Local Usage**
1. Clone the repository:
```bash
git clone https://github.com/ansilcode-me/personal-ai-bot.git
cd personal-ai-bot

Install dependencies:
pip install -r requirements.txt

Create a .env file based on .env.example:
BOT_TOKEN=your_telegram_bot_token
OPENAI_API_KEY=your_openai_api_key

Run the bot:
python bot.py
